# Docker Web Deployment Demo

## Overview
This project demonstrates containerized deployment of a static website using Docker and Nginx.

## 项目介绍
该项目用于演示如何使用 Docker 和 Nginx 部署静态网页，实现快速启动与运行。

---

## Tech Stack
- Docker
- Docker Compose
- Nginx
- Linux

## 技术栈
- Docker
- Docker Compose
- Nginx
- Linux
